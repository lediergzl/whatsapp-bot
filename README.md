
# Bot de WhatsApp

Este bot de WhatsApp permite gestionar grupos y enviar resultados de lotería.

## Opciones de Despliegue Gratuitas (Sin Tarjeta)

### Opción 1: Glitch.com
1. Crea una cuenta en Glitch.com (solo necesitas correo electrónico)
2. Crea un nuevo proyecto importando desde GitHub
3. Importa este proyecto
4. El bot se iniciará automáticamente

### Opción 2: Railway.app con GitHub Student
1. Regístrate con tu cuenta de GitHub
2. Si tienes GitHub Student Pack puedes obtener créditos gratuitos
3. Crea un nuevo proyecto desde GitHub
4. Configura el comando de inicio como `npm start`

### Opción 3: Replit (usando siempre)
1. Mantén tu navegador abierto con la pestaña de Replit
2. Usa el botón "Run" para iniciar el bot
3. Escanea el código QR desde WhatsApp

## Comandos disponibles
- `!cerrar` - Cierra el grupo (solo administradores)
- `!abrir` - Abre el grupo (solo administradores)
- `!tarjeta` - Envía la tarjeta de resultados (solo administradores)
